Программа по работе с датчиками в проекте

Данная программа позволяет формировать лог датчиков выбранного проекта.

Настройки проекта

При запуске програмы загружается проект из конфигурационного файла по умолчанию. Конфигурационный файл по умолчанию default.conf находится в директории с программой.
В меню "Файл"->"Открыть..." можно загрузить настройки проекта из выбранного конфигурационного файла.
В меню "Файл"->"Сохранить..." можно сохранить настройки проекта в выбранный конфигурационный файл.

Поиск датчиков

В поле "Имя датчика" указывается имя искомого датчика.
В поле "Проект" указывается путь к директории с проектом.
По кнопке "Поиск датчиков" происходит поиск датчиков в проекте. Поиск происходит как по всем файлам указанного проекта, так и только по отмеченным файлам.

Запись лога датчиков

В поле "Файл с датчиками" указывается путь к файлу с логом датчиков.
В поле "Размер результирующего файла" указывается требуемый размер файла с логом датчиков.
По кнопке "Записать лог датчиков в файл" происходит дозапись лога датчиков.

Дерево проекта

В разворачивающемся списке файлов проекта отмечаются те файлы и директории, в которых необходимо искать датчики, которые в дальныйшем буду записываться в результирующий лог.

Лог

Результаты работы программы выводятся в программный лог. Более детальный лог работы программы выводится в файл.

Файлы с отладочной информацией

Все файлы с отладочной информацией записываются в директорию с программой.

Название файлов:

  • Детальный лог программы: log.txt
  • Список датчиков, найденных во всех файлах проекта: sensors-from-all-files.txt
  • Список датчиков, найденных только в отмеченных файлах проекта: sensors-from-checked-files.txt
  • Отладочная информация о записи лога датчиков: trace-sensor-log-writer.txt

Основные принципы работы программы

При поиске датчиков в проекте учитываются все найденные датчики.

При записи датчиков в лог удаляются дублирующиеся датчики, датчики из блока else (при определенных условиях).
Для изменения фильтра выводимых датчиков см. класс SensorLogWriter функцию readSensors.
